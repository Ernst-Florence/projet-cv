Meine Fähigkeiten

HTML/CSS

HTML (Hypertext Markup Language) ist eine Auszeichnungssprache, die zur Strukturierung von Inhalten auf Webseiten verwendet wird, während CSS (Cascading Style Sheets) die Gestaltung und das Layout dieser Inhalte steuert, indem es das Erscheinungsbild definiert.

Javascript

JavaScript ist eine Skriptsprache, die in Verbindung mit HTML und CSS für die Interaktivität auf Webseiten sorgt, indem sie Inhalte dynamisch ändert und auf Benutzeraktionen reagiert.

MySQL/PHP

PHP ist eine serverseitige Skriptsprache, die in Verbindung mit MySQL-Datenbanken verwendet wird, um dynamische Webseiten zu erstellen. PHP verarbeitet die Anfragen, generiert HTML und interagiert mit MySQL, um Daten zu speichern und abzurufen.
